# cli

快速为模块和应用程序创建一个开发环境

## 起步

安装：

```bash
npm install -g @fe6/cli
# OR
yarn global add @fe6/cli
```

创建一个项目：

```bash
fe6 init my-project
# OR
fe6 init a -c https://github.com/iq9891/ieditor.git
```
