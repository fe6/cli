# {{name}}支付宝小程序

## 准备

1. 安装 NodeJS 13+
2. 选择安装 [yarn](https://classic.yarnpkg.com/zh-Hans/)
3. 下载[支付宝小程序开发者工具](https://opendocs.alipay.com/mini/ide/download)

## 调试项目

1. `npm install` 或者 `yarn`
2. 打开支付宝小程序开发者工具

## 上传代码

1. 修改 `api.js` 中环境变量文件
2. 支付宝小程序开发者工具 上传代码
